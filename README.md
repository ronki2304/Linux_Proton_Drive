# ProtonDrive Linux Client

A self-contained CLI client for ProtonDrive on Linux.

## Requirements

- [Bun](https://bun.sh) ≥ 1.1

## Install dependencies

```bash
bun install
```

## Build binary

```bash
bun build --compile src/cli.ts --outfile dist/protondrive
```

## Run

```bash
./dist/protondrive --version
```
