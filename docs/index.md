# ProtonDrive Linux Client — Documentation Index

**Type:** Monolith CLI

**Architecture:** Layered CLI (commands → core → sdk)
**Last Updated:** 2026-04-05

## Project Overview

A self-contained command-line client for ProtonDrive on Linux. Supports authentication (SRP protocol), file upload/download with PGP encryption, bidirectional sync with conflict detection, and status reporting. Compiles to a single binary with no runtime dependency.

## Quick Reference

- **Tech Stack:** TypeScript 5 + Bun 1.3.x + Commander 14 + @protontech/drive-sdk + openpgp v6
- **Entry Point:** `src/cli.ts`
- **Architecture Pattern:** Commands → Core → SDK (four-layer)
- **State Storage:** `bun:sqlite` (local SQLite file)
- **Credential Storage:** OS keychain via `@napi-rs/keyring`

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) ≥ 1.1

### Setup

```bash
bun install
```

### Build Binary

```bash
bun build --compile src/cli.ts --outfile dist/protondrive
```

### Run Tests

```bash
bun test                              # Unit tests
bun test src/__e2e__/                 # E2E (requires built binary)
bun test src/__integration__/         # Integration (requires Proton credentials)
```
